# Exam Portal Documentation

## Overview
The Exam Portal is an online platform that facilitates the management and administration of examinations. It allows users to create, schedule, take, and manage exams efficiently.

## Features
- **User Management:** Register and manage users, including students and instructors.
- **Exam Creation:** Create and configure multiple choice, true/false, and open-ended exams.
- **Scheduling:** Schedule exams for specific date and time.
- **Grading:** Automated and manual grading options with feedback.
- **Reports:** Generate reports for users and exams.

## Getting Started
To get started with the Exam Portal, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/jadu918/exam-portal.git
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

## Usage
1. Log in with your credentials.
2. Navigate to the dashboard to access different features.
3. Follow the prompts to manage exams.
